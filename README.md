# The Creative Producer

<img src="https://i.imgur.com/D1piDh6.jpeg" alt="logo" width="500"/>

## _AI-powered image + video creation, website generating and Instagram posting for streetwear brands_

Generate streetwear visuals, turn them into Reels, and post directly to Instagram Business accounts — all from one dashboard.
IBM Bob is used to generate captions and webpage postings.



[![Watch the video](https://youtube.com)](https://youtu.be/lX3BVY-eX1U)



*Download & Install IBM Bob*

|[IBM Bob](https://bob.ibm.com/)|

*Create Free Trial for IBM Bob*

*Clone and open the starter project*

Follow these steps to clone the repository for thIS project:

In IBM Bob, click Terminal > New Terminal.

In the terminal, copy and paste these commands to clone the starter project and open it:

```bash
git clone https://github.com/filxy/Creative-Producer.git
```


```bash
cd Creative-Producer
```

+ *Open Project folder in IBM Bob.*

+ *Update New Theme to HTML page.*

+ *Upload images and audio to cloud to share URL in .env file with IBM Bob.*

+ *Prompt IBM Bob to update the webpage by adding new images.*

+ *Create a REEL of the image, add transition's and add music.* 

+ *Upload to Instagram account with Captions.*

+ *Add Instagram auto-posting to this project.*

### Note: 

Instgram Business account is required and INSTAGRAM API and INSTAGRAM ID needs to be updated in .env file along with Media URL'S.

You can use imgur to upload images or Uploadcare and share URL's links of any media.

Important: In IBM Bob, click File > Open Folder, then open the Creative-Producer folder that you just cloned.


*Features*
*1. AI HTML Generation*
- Generate stylish product images: hoodies, streetwear, neon effects
- Custom colors, backgrounds, and styles
- Save/download for product catalogs

*2. AI Video Creation*
- Animate product images into 5-10s Reels
- Auto-add transitions, music, and text overlays
- Export in IG-ready 9:16 format

*3. Instagram Business Posting*
- OAuth login with Instagram Business API
- Schedule and publish Reels + Posts directly
- Track media status and analytics

*4. Video Hosting*
- Auto-upload to Web and Instagram Reels.
